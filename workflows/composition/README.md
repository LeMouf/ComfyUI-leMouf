# Composition Workflows

Release line: `0.3.2`

## Available master workflows

- `composition_studio_master_0-1-0.json`
  - Single-step composition workflow (`role=composition`).
  - Opens Composition Studio directly from pipeline run.
  - Supports optional `resources_json` input on `LoopPipelineStep` to preload assets.
  - Profile: `tool` / `master`.
